Hg
-------------

另外,细心的读者,可能已经发现了, `大妈`_ 使用 `Hg` 进行版本管理!

为毛? 严正推荐参考:

- `EPE-374`_ ~ Choosing a distributed VCS for the Python project (综合对比了 svn/hg/git/bzr)
- `[翻譯]Git 與 Mercurial 的分析 | TWPUG::Kiang <http://blog.twpug.org/416>`_

对于 `大妈`_ 而言,最中意的应该算~ 这是所有版本管理命令中'''最短'''的好不?!
